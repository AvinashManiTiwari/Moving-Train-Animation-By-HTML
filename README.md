# 🚆 Moving Train Animation Using HTML

## 🚀 Live Demo

👉 https://AvinashManiTiwari.github.io/Moving-Train-Animation/

## 🌟 Project Overview

A creative and beginner-friendly web project that simulates a moving train using pure HTML. This project demonstrates basic animation concepts, webpage structuring, and visual design techniques.

## ✨ Features

* 🚆 Moving Train Animation
* 🎨 Attractive Visual Design
* 🌐 Pure HTML Implementation
* 📱 Beginner-Friendly Project
* 💡 Easy to Understand and Modify
* 🚀 Hosted on GitHub Pages

## 🛠️ Technologies Used

* HTML5
* Font Awesome

## 🎯 Learning Outcomes

* HTML page structure
* Navigation and linking
* Image integration
* Basic animation concepts
* GitHub Pages deployment
